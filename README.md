# PlaylistApp

## 📖 Overview
This repository is a demo application for creating and managing a custom music playlist.  
It demonstrates dynamic UI interactions with **React** and **Material UI**, and automated UI testing with **Playwright**.  
The app showcases playlist creation, track management, and total duration calculation.
## 🚀 Getting Started

### Install dependencies
```bash
npm install
```
This command installs the necessary dependencies specified in the package.json file, including Playwright and other related packages.

## Configuration
```
add BASE_URL value
```
Playwright configurations are stored in the `playwright.config.ts` file.

## Run all Tests

```bash
npm run test
```
Custom script to run test. They can be add in the `package.json` file for different browsers or test configurations.
